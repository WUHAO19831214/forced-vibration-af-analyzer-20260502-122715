# 实验流程

## 设备与准备

- 固定摄像头和振动装置；
- 白色哑光背景与红色光斑；
- 与光斑平面共面的长度标尺；
- 驱动器、放大器和独立频率验证设备；
- 如需理论比较，另备力、位移或加速度参考传感器。

## 浏览器准备

使用 HTTPS 在线页面或 localhost，授权摄像头。记录浏览器、操作系统、摄像头分辨率、标称/实测帧率，并尽量锁定曝光、对焦和白平衡。

## 标定

1. 确认光斑稳定锁定；
2. 将标尺放在光斑所在平面；
3. 绘制实际 1 cm 线段；
4. 记录显示的 pixel/cm 关系；
5. 摄像头或光路移动后重新标定。

## 驱动与扫频

1. 用独立设备确认驱动器能覆盖目标频段；
2. 设置起始、峰值、步进和停留时间；
3. 每个频率点应停留足够多个周期；
4. 频率切换后先等待瞬态衰减，再记录稳态窗口；
5. 记录升频和降频方向；
6. 每个频率至少重复 3 次。

旧程序不会自动舍弃瞬态，也不会按周期数控制记录窗口，操作者必须在实验设计中处理。

## 数据记录

同步记录间隔是表格写入间隔，不是视频采样率。建议不要在同一频率下盲目产生大量重叠窗口；正式分析应同时保存原始视频或逐帧数据。

## 建议质量检查

- 背景无其他红色区域；
- 光斑不过曝、不离开画面；
- 每个窗口至少包含足够帧数和周期数；
- 标尺与光斑共面；
- 实际驱动频率与设定值一致；
- 驱动力幅值在扫频过程中可控；
- 异常窗口有排除理由。

## 结果报告

至少报告：装置结构、光路、驱动设备、摄像头、浏览器、标定长度和像素值、频率范围、步进、停留时间、记录间隔、重复次数、原始/排除数据、不确定度和软件 commit。

界面中的 `A` 应在论文中改写为“光斑峰—峰位移范围”，除非另行完成单边振幅和光路反演。
