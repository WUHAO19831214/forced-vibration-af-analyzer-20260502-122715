# 物理模型

## 理想线性受迫振动

单自由度黏性阻尼系统可写为：

```text
m x'' + c x' + k x = F0 sin(ωt)
```

稳态位移振幅为：

```text
X(ω) = F0 / sqrt((k-mω²)² + (cω)²)
```

相位滞后可写为：

```text
φ(ω) = atan2(cω, k-mω²)
```

固有角频率 `ω0=sqrt(k/m)`，无阻尼固有频率 `f0=ω0/(2π)`。有阻尼时响应峰值位置取决于阻尼。

## 当前程序与模型的对应关系

| 理论量 | 程序中可用的信息 | 限制 |
|---|---|---|
| 驱动频率 `f` | Web Audio 设定频率 | 未独立实测 |
| 驱动力幅值 `F0` | 无直接测量；仅有音量增益 | 不可视为恒定机械力 |
| 位移 `x(t)` | 红色光斑重心 `y(t)` | 是光斑坐标，不是本体位移 |
| 响应振幅 `X` | 程序实际用 `max(y)-min(y)` | 是峰—峰范围，未除以 2 |
| 相位 `φ` | 无 | 未实现同步相位估计 |
| `m,c,k` | 无 | 未估计 |

因此，当前图像是“设定频率—光斑位移范围”经验曲线，不能直接等同于理论 `X(ω)`。

## 从光斑范围到机械振幅

若光斑由反射镜小角度转动产生，常见小角度近似为反射光偏转角约为镜面转角的 2 倍。屏幕距离、振动部件几何和安装方式还会引入比例关系。完整反演至少需要：

1. 光路传播距离；
2. 入射与反射几何；
3. 反射镜转角与本体位移关系；
4. 光斑峰—峰范围到单边振幅的转换；
5. 独立标定和不确定度。

当前程序只完成画面内 cm/pixel 标定，没有这些光路和结构参数。

## 建议的理论比较方法

### 归一化曲线

当驱动力幅值无法可靠测得时，可先比较：

```text
R(f) = Δy(f) / max_f Δy(f)
```

这只能比较曲线形状，不能得到绝对机械柔度。

### 参数拟合

若已知 `F0` 且 `Δy` 已反演为机械单边振幅，可对 `m,c,k` 或 `f0,ζ,scale` 做非线性最小二乘拟合。应报告参数区间、残差和模型适用性。

### 峰值比较

比较实验峰值频率和理论峰值频率时，应考虑频率步进造成的离散误差、扫频方向迟滞、瞬态未衰减和驱动幅值随频率变化。

## 边界条件

理论模型要求线性、小振幅、参数稳定、单自由度、正弦稳态驱动。摩擦、非线性刚度、碰撞、多模态、执行器饱和和光路几何非线性都会破坏该模型。
